mp3 tag to unicode converter
I have a beautiful Nokia phone with media player, but also I got a problem. I can't see Cyrillic song names on it.
I found that I have to convert all mp3 tags to unicode.
After research, I found a simple program that could convert all your mp3 tags to unicode (version 2).

It's a command line tool written in java (so it requires java).
Also I create a small batch that you can use to simplify your work with this tool
Just download the archive, extract it, and start the batch id3iconv.cmd


NOTE: If you know what you are doing, You can modify id3iconv.cmd to specify the path to java executable
and ansi encoding of your mp3 files if this encoding is different from your system settings.
